BandMap

BandMap is a way to explore Bandcamp on a Map. Based on a user's genre and location preferences, BandMap creates an interactive map where they can find links and/or previews to the artist's music. Also, there is a feature that allows the user to create a Spotify playlist of the artists from a selected city.
